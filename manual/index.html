    Create the track and data type

    First you need to create custom data type for your track, create new class that inherit from Feature ,

    then create a new class inherit from Track<T> where T should be the custom data type you just created.

    Create the editor GUI for newly created track

    Create a new class inside 'Editor' folder inherit from FeatureGUI<T> where T should be the custom data type you just created.

    Make sure newly generated beat-map will contain this track

    Go to CreateBeatmap() and add your newly created track accordingly.

    Hook up with event provider

    Use Register<T>(Action<T>) and Unregister<T>(Action<T>) to link event callback to the music timeline pointer event so the note generator could generate note using the newly created track and its data.
